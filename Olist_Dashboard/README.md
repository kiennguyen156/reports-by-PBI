# 🛒 Olist E-Commerce Business Dashboard

A comprehensive Power BI dashboard designed to analyze the business performance of Olist, a Brazilian e-commerce and retail-tech platform. This report provides deep insights into sales trends, customer behavior, seller performance, and customer satisfaction.

## 📊 Key Features
- **Business Overview:** High-level summary of total revenue (R$ 16.01M), transaction volumes, and payment behaviors.
- **Customer Behavior:** Deep dive into RFM Segmentation (Recency, Frequency, Monetary) and seasonal trends.
- **Sales Analysis:** Tracking sales performance, profit margins (~85%), and geographic distribution across Brazilian states.
- **Seller Performance:** Monitoring active sellers growth, delivery times, and cancellation rates.

## 📁 Tools & Technologies
- **Power BI:** Data modeling, DAX, interactive dashboard design.
- **Power BI Service:** Cloud deployment and stakeholder sharing.
- **Power Query:** ETL workflows and data shaping.
- **Analytical Frameworks:** RFM Analysis, Sentiment Analysis (Word Cloud).
